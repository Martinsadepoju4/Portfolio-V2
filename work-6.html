A IP address Tracker.

About the Project.

I single handedly took it upon myself to create an IP address tracker with the use of the geolocation API, Node.js, and EJS to make it happen. I wanted to learn more about the backend side of things, so I got started. With Node.js, I built the basic part of the tracker, making sure it could handle requests and talk to the IP Geolocation API. This API was like the brains of the tracker, giving it lots of info about IP addresses. Then, I used EJS to make the tracker look good on the front end. People could type in an IP address and quickly find out where it's from and other details. It was a lot of work to get everything just right, but I'm proud of how it turned out. This project taught me a lot about APIs, Node.js, and EJS, and it made me even more excited about web development.
